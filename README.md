### Troubleshooting

**If training is too slow:**
- Use Google Colab with GPU
- Reduce number of epochs (minimum 10)
- Reduce batch size if out of memory

**If you get memory errors:**
- Reduce batch size to 16
- Reduce hidden dimensions to 256
- Close other applications

**If BERT score installation fails:**
```bash
pip install bert-score --no-deps
pip install transformers torch
```

### What the Code Does

1. **Loads data** from CSV files
2. **Builds vocabularies** for Sanskrit and English
3. **Creates model** with encoder-decoder + attention
4. **Trains** for 15 epochs with early stopping
5. **Evaluates** on dev and test sets
6. **Generates** submission.csv with predictions
7. **Calculates** BLEU, BERTScore, and efficiency metrics
8. **Shows** translation examples with error analysis
